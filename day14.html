adding DOM elements with javascript

It is possible to add new elements to the DOM with javascript. this can be done with the following methods

  • document.createElement()
  • document.createTextNode()
  • element.appendChild()
  • element.insertBefore()

Example

            
                // create a new paragraph element
                var p = document.createElement('p');
                // create a new text node
                var text = document.createTextNode('This is a new paragraph');
                // append the text node to the paragraph element
                p.appendChild(text);
                // append the paragraph element to the body of the document
                document.body.appendChild(p);

Remove DOM elements with javascript

It is possible to remove elements from the DOM with javascript. this can be done with the following methods

  • element.removeChild()
  • element.remove()

Example

            
                // get the paragraph element
                var p = document.querySelector('p');
                // remove the paragraph element from the body of the document
                document.body.removeChild(p);
